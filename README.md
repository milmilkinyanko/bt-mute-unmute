# Mute/Unmute Script with Bluetooth Button

## Features
- Amazonで買った300円Bluetoothボタンを使ってマイクをミュート/アンミュートするスクリプト
- amixerを使ってマイクのインプットボリュームをいじることでどんなチャットツールであってもミュート/アンミュート可能

## Requirements
- python3
- evdev (pip module)
- ALSA
